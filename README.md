# Dynamic Milk Pricing Replication
 data and replication code for 2025 fluid milk dynamic pricing paper

Author: Aaron Adalja

Date: December 2025

The files contained herein allow for replication of the results for:
"Dynamic pricing to reduce retail dairy shrink: Evidence from lab and grocery store experiments"
by Aaron Adalja, Aljosa Trmcic, Saumya Desai, Yixuan Wang, Leonie Kemmerling, Martin Wiedmann 

root folder:

Milk_pricing_study1_replication.R - This is the R code file that is used to process the data and generate all tables, figures, and statistics for Study 1.

Milk_pricing_study2_replication.R - This is the R code file that is used to process the data and generate all tables, figures, and statistics for Study 2.

data folder:

rawdata_study1.csv - raw experiment data for Study 1 

rawdata_study2.xlsx - raw sales data for Study 2
